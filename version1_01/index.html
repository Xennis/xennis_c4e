    Welcome

    Welcome to my website! Explore the site: You can change the background image and colors ... resize, drag and minimize the articles ... and much more. This is not a old static site, so you can do a lot of interactive things!

    Have fun!

    Requirements

    You need the latest version of your web browser to enjoy the beauty of this web design! Older versions do not support new web standards like HTML5 or CSS3. You will get better results (speed, animation, font, ...) if you use Firefox, Chrome, Opera, Safari, etc. and not Internet Explorer.

    Please report all bugs.

    About

    This site is written in HTML5 and CSS3. I used jQuery (v1.5.1) and jQuery user interface (v1.8.11) as JavaScript framework.

    jQuery plugins and tutorials

    • QueryLoader (v1.0) written by Gaya Kessler
    • Fullscreenr (v1.0) written by Jan Schneiders
    • Animated Menu published by jQueryGlobe
    • jScrollPane (v2.0.0.beta9) written by Kelvin Luck
    • Mousewheel (v3.0.4) written by Brandon Aaron
    • AnythingZoomer (v1.0) written by Chris Coyier
